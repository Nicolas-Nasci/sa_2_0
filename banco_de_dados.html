O que é um Banco de Dados

Um banco de dados é uma coleção estruturada e organizada de informações, geralmente armazenada
de forma eletrônica em sistemas computacionais. Sua gestão é realizada por um Sistema de
Gerenciamento de Banco de Dados (DBMS). O conjunto formado pelos dados, o DBMS e os aplicativos
relacionados é conhecido como sistema de banco
de dados, comumente referido apenas como banco de dados.

Banco de Dados Relacional e NÃO Relacional

Um banco de dados relacional é um conjunto de informações que organiza dados em relações
predefinidas, em que os dados são armazenados em uma ou mais tabelas (ou "relações") de colunas
e linhas, facilitando a visualização e a compreensão de como diferentes estruturas de dados se
relacionam.
Um banco de dados não relacional é aquele que não utiliza o esquema tradicional de tabelas com
linhas e colunas, característico dos sistemas de banco de dados convencionnal

Diagrama Conceitual DER

O Diagrama de Entidade e Relacionamento (DER) é uma representação gráfica do Modelo de Entidade
e Relacionamento. Ele evita a abstração excessiva, apresentando informações relevantes de forma
mais visual e intuitiva, facilitando a compreensão.

Diagrama Lógico MER

De maneira simples, o Modelo de Entidade e Relacionamento (MER) é um diagrama utilizado para
projetar Bancos de Dados Relacionais, baseado nas relações entre objetos reais, sendo
representado por entidades e seus respectivos relacionamentos.

Diagrama DER do Exercício

Diagrama de Entidade e Relacionamento (DER) é uma representação gráfica do Modelo de Entidade
e Relacionamento. Enfim, isso evita excesso de abstração, trazendo para a realidade informações
pertinentes de uma forma mais visual e, consequentemente, mais intuitiva.
Um diagrama ER é um modelo que representa as relações físicas entre as entidades. Ele possibilita
visualizar como as entidades estão interconectadas, como seus relacionamentos funcionam e onde
é possível aprimorar essas interações.

O que é dicionário de dados?

Um Dicionário de Dados é fundamental para fornecer respostas sobre os dados em um sistema.
Trata-se de um documento que contém informações detalhadas sobre os dados em um banco
de dados, incluindo descrições de tabelas, colunas e o significado de cada elemento.
